# VSTH



# 1. Code structure
```
├─Admin        # ORM models defination.
├─Common
├─conf         # Configure files.
├─ctrls        # Controller objects.
├─doc
├─logs
├─routers
├─static       # Js files, css files, images etc.
├─tools
├─vendor       # Third part packages used in VSTH.
└─views        # HTML files.
```


# 2. How to compile

go build main.go
